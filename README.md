# 數智入口\_前端

## 專案結構

```
├── public/                 # 靜態資源目錄
│   ├── favicon.ico        # 網站圖標
│   └── assets/           # 靜態資源文件
├── src/                   # 源代碼目錄
│   ├── api/              # API 接口定義
│   ├── assets/           # 項目資源文件
│   ├── components/       # 公共組件
│   ├── layouts/          # 布局組件
│   ├── router/           # 路由配置
│   ├── stores/           # Pinia 狀態管理
│   ├── styles/           # 全局樣式
│   ├── views/            # 頁面組件
│   ├── App.vue           # 根組件
│   └── main.ts           # 入口文件
├── .vscode/              # VS Code 配置
├── .eslintrc.js         # ESLint 配置
├── .prettierrc          # Prettier 配置
├── index.html           # HTML 模板
├── vite.config.ts       # Vite 配置
├── tsconfig.json        # TypeScript 配置
└── package.json         # 專案依賴配置
```
